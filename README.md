## 🎮 게임 소개

**아케인 배라지(Arcane Barrage)**는 끝없이 몰려오는 적성 몬스터들을 상대로 강력한 마법 투사체와 스킬을 사용하여 끝까지 생존하는 **웹 기반 로그라이트 서바이벌 게임**입니다. (뱀파이어 서바이벌 류)

전투를 통해 경험치를 획득하고 레벨업을 하여 나만의 화려한 마법 트리(Magic Tree)를 조합해 보세요! 가시 갑옷으로 데미지를 반사하고, 유도 미사일로 남김없이 적을 분쇄하세요.

---

## 🕹️ 조작 방법 (Controls)

*   **이동:** `W`, `A`, `S`, `D` 또는 `방향키 (Arrows)`
*   **일시정지 및 메뉴 열기:** `ESC`
*   **UI 조작:** `마우스 클릭`, 방향키 및 `Enter`키 지원

---

## ✨ 핵심 기능 및 특징

1.  **끝없는 마법 탄막 (Bullet Hell)**
    *   투사체 수 증가, 유도 미사일 폭발, 주변을 회전하는 마법 구슬, 화염 오라 등 다양한 마법 효과!
    *   화면을 가득 채우는 마법 스킬 조합 시너지 경험.
2.  **RPG 성장 시스템**
    *   적 처치 시 드랍되는 경험치(XP) 보석을 흡수하여 레벨업.
    *   레벨업 할 때마다 무작위로 제시되는 3장의 스킬 카드 중 하나를 선택.
3.  **난이도 스케일링 체계**
    *   라운드 타이머에 따라 점차적으로 강력하고 거대한 적 등장.
    *   스테이지 클리어를 위한 **보스(BOSS) 몬스터** 시스템.
4.  **관리자 기능 (Admin Console)**
    *   메인 화면에서 관리자(`admin`/`1234`)로 로그인 시, 테스트용 치트 패널 기능 활성화!

## 📜 업데이트 내역 (Changelog)

### v1.5.0 (실시간 분석 및 방문자 통계 업데이트)
*   **실시간 접속자 카운트:** Firebase Realtime Database 연동을 통해 현재 게임을 플레이 중인 인원수를 실시간으로 메인 화면에 표시
*   **누적 방문자 시스템:** 중복되지 않는 세션 방문자 수를 집계하여 총 누적 방문자 지표 제공
*   **Google Analytics 4 통합:** 상세 접속 로그 및 유저 행동 분석을 위해 GA4(G-MNYM1N6G7B) 분석 코드 삽입
*   **인게임 패치 노트 섹션:** 메인 화면 하단에 최근 업데이트 내역을 바로 확인할 수 있는 UI 영역 추가

### v1.4.0 (스킬 레벨 및 쿨타임 UI 업데이트)
*   **보유 스킬 레벨 표시:** 좌측 하단 패널에 현재 보유 중인 모든 스킬의 아이콘과 레벨(`Lv.n`) 출력 기능 추가
*   **스킬 분류 시스템:** 스킬을 '액티브(Active)'와 '패시브(Passive)'로 분류하여 시각적으로 분리된 패널 제공
*   **실시간 쿨타임 게이지:** 액티브 스킬 아이콘 하단에 충전 상태에 따라 색상이 변하는 동적 쿨타임 게이지 바 구현
*   **UI 레이아웃 최적화:** 패널 간 간격 및 위치 조정을 통해 가독성 및 UI 시인성 개선

### v1.3.0 (모바일 최적화 및 쥬스 편의성 업데이트)
*   **시각 효과 강화 (Juice):** 적중 시 데미지 텍스트 렌더링, 강력한 체력 타격 및 미사일 폭발 시 화면 흔들림(Screen Shake) 효과 추가.
*   **최고 기록 시스템:** 브라우저 'localStorage' 연동을 통해 게임 오버 시 나의 역대 최대 '생존 시간'과 '최대 킬 수' 저장 및 메인화면 출력
*   **모바일 가상 조이스틱:** 터치 디스플레이용 다이나믹 가상 조이스틱 이동 조작 추가

### v1.2.0 (BGM 시퀀서 및 단축키 시스템)
*   **웹 오디오 API BGM 시퀀서 구축:** 판타지 전투 테마 곡(A단조) 하드코딩. (멜로디, 베이스, 드럼 3트랙 재생)
*   **상단 바 오디오 믹서 제어:** BGM 온/오프(`B`), 효과음 온/오프 토글 버튼(`N`) 연동 및 단축키 할당
*   **메뉴 키보드 단축키 지원:** 오버레이 메뉴 창 내부 `ArrowKeys`, `Enter`, `ESC(메뉴 온오프)` 완벽 지원

### v1.1.0 (사운드 시스템 업데이트)
*   **Web Audio API 기반 8비트 사운드 추가:** 외부 오디오 에셋(mp3 등) 리소스 없이 코드로 `audio.js` 플러그인 독자 도입
*   **인게임 효과음 반영:** 투사체, 타격, 피격, 레벨업, 폭발 등 맞춤형 SFX 구현

### v1.0.0 (최초 배포 버전)
*   **코어 플레이 루프 완성:** 적 생성, 플레이어 조작, 경험치 시스템 기능 도입
*   **13종의 다양한 마법/패시브 스킬 추가:** 이동속도, HP 증가, 자연 회복부터 번개, 운석, 가시 갑옷, 흡혈 등 특수 기능 스킬 완비
*   **거대 운석 스킬 리뉴얼:** 무작위 폭발 대신 적을 직접 추적 후 광역 폭발을 일으키는 **[미사일 발사]**로 스킬 개편
*   **UI/UX 구축:** 깔끔하고 다크한 판타지 테마의 CSS UI 구축 및 오버레이 메뉴 시스템 적용
*   **어드민 시스템:** 배포용 환경과 개발 테스트를 분리하기 위한 관리자 모드와 치트 패널(강제 레벨업[U], 경험치 흡수[E], 모든 적 제거[R] 등) 적용
